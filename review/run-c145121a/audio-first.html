Equal Circles, Equivalent Fractions

Keep this example visible

1/2 = 2/4

  • Equal whole: both circle boundaries are the same size.
  • Equal parts: each circle is divided into regions of equal area.
  • Numerator: the top number counts shaded parts.
  • Denominator: the bottom number counts all equal regions, not partition lines.
  • Same shaded area: both models cover one half of the whole.
Only compare shaded areas when the circles represent same-sized wholes.
