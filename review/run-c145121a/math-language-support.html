Equivalent Fractions with Equal-Sized Circles

Build, compare, subdivide, and explain circle models. Different fraction names can describe the same area of one whole.

Lesson goal

Recognize and generate equivalent fractions by showing that they cover the same shaded area of equal-sized wholes.

Compare shaded areas only when both circle boundaries represent the same-sized whole. Count equal enclosed regions—not partition lines.
